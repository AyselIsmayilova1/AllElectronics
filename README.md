# Amplifier — Physics Guide & Practical Insights

A 5-slide Physics Guide on electronic amplifiers — covering the definition, purpose,
core components, working principle, common applications, and conclusions.
Designed as a lesson presentation and practical insight resource for a physics / electronics module.

**Authors:** Xumar Bağıyeva · Aysel İsmayılova
**Verified by:** Telman Askeraliyev — Physics Teacher, Azerbaijan, Baku (Fizika Muellimi)

---

## Topics Covered

- Amplifier definition — increasing signal amplitude without altering frequency or shape
- The challenge of weak signals (millivolt-level from microphones, pickups, smartphones)
- Signal integrity: linearity, clipping, and distortion avoidance
- Key components: transistor/vacuum tube (the active "gatekeeper"), power supply (the energy reservoir), and heatsinks (thermal management)
- Working principle: the water-tap analogy — small input controls large power output
- Common applications: home audio, instrument amps, PA systems, car audio
- Conclusion: `Small Signal + External Power = Amplified Output`
- Evolution: from vacuum tubes → transistors → Class D digital amplification

# Field Effect Transistors (FETs) — Comprehensive Technical Report

A 6-slide comprehensive technical report on Field Effect Transistors — covering their
structure, operating principle, types (JFET vs MOSFET), enhancement/depletion modes,
modern applications, and biasing techniques.

**Authors:** Xumar Bağıyeva · Aysel İsmayılova
**Verified by:** Telman Askeraliyev — Physics Teacher, Azerbaijan, Baku (Fizika Muellimi)

---

## Topics Covered

- FET definition — unipolar, voltage-controlled device using a single charge carrier type
- Physical structure and terminals: Source (S), Drain (D), Gate (G), and the channel
- The "Field Effect" operating principle — electrostatic field controls channel conductivity
- High input impedance advantage — Gate draws near-zero current
- JFET vs MOSFET architecture differences and use cases
- Enhancement Mode (E-MOSFET) vs Depletion Mode (D-MOSFET) logic states
- Deep dive: Threshold voltage, operating regions (Ohmic, Saturation), pinch-off effect
- CMOS technology — zero static power consumption, the backbone of modern computing
- Performance metrics: switching speed, thermal runaway protection, ESD sensitivity
- Specialized types: Power MOSFETs, MESFETs (GaAs / 5G), FinFETs (3nm/5nm chips)
- Biasing techniques: Fixed Bias, Self-Bias, Voltage Divider Bias

---

## Key Equations

| Quantity | Formula |
|----------|---------|
| Shockley equation (JFET) | `I_D = I_DSS × (1 − V_GS / V_P)²` |
| Threshold voltage (MOSFET) | `V_GS(th)` — minimum Gate voltage to open channel |
